# HALO Workflow Blog

AI Agent 시대의 워크플로우 엔지니어링에 관한 기술 블로그 시리즈.

**HALO Workflow**(Harness · Agentic · Loopback · Orchestration)의 철학과 원리를 다룹니다.

## 시리즈 목차

1. **AI Agent의 명확한 한계** — Needle in a Haystack, Car Wash Test, Blind Context
2. **왜 한계가 발생하는가** — VRAM, KV Cache, Context Rot의 물리적 제약
3. **Prompt → Context → Harness 엔지니어링** — 세 단계의 진화
4. **나눠서 정복한다** — Phase 시스템과 우공이산의 원리
5. **실행이 10분이 된 세상** — 개발자의 마인드셋
6. **HALO Workflow** — 철학이 구조가 되다
